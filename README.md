# LIFE OS v2 — TAKUMI's Cognitive Workspace

LIFE OS v2 は、Notionを唯一の操作起点とし、GTD（Getting Things Done）とバレットジャーナル（Bullet Journal）の思想を融合させた、TAKUMI専用の日常・ビジネス統合型コグニティブワークスペースです。

毎朝7:00に自動でデイリーノートが生成され、その日の予定、タスク、プロジェクト状況がすべて1ページに集約されます。また、毎晩22:00にはその日のタスクが自動でスキャン・整理され、未完了のタスクは翌日に引き継がれます。

---

## 🚀 システムの特徴

1. **唯一の操作起点（デイリーノート）**
   - 毎日1ページだけ開けば、その日のすべて（予定・タスク・PJ・メモ）が完結します。
   - 複雑なDBを直接操作する必要はありません。

2. **GTD × バレットジャーナル**
   - `Capture`（LOGに何でも書く）→ `Clarify`（SORAが夜間にタスクDBに自動登録）→ `Engage`（TODAYセクションで実行）。
   - 未完了タスクの翌日への自動引き継ぎ（Migration）を自動化。

3. **Googleカレンダー双方向連携**
   - 当日のイベントが自動でデイリーノートの `SCHEDULE` セクションに表示されます。
   - ミーティングイベントは、アジェンダやAI Meeting Note用の専用サブページが自動生成され、スケジュールから1クリックで遷移できます。

## 📁 リポジトリ構成

- `README.md` — 本ドキュメント（概要と使い方）
- `DESIGN.md` — システム詳細設計書（アーキテクチャ・自動化フロー）
- `SCHEMA.md` — Notion データベーススキーマ定義
- `FLOW.md` — 朝・夜の自動化プロセス詳細
- `scripts/` — 自動化スクリプト群（Python）
  - `morning_daily_generator.py` — 朝7:00のデイリーノート生成
  - `evening_task_scanner.py` — 夜22:00のタスクスキャン・同期
  - `life_os_runner.py` — 統合ランナー
